# Prerequisite Concepts

Concepts and background knowledge assumed by the paper. Each entry gives a brief
explanation and a pointer to learn more.

## Joint-Embedding Predictive Architecture (JEPA)

**What it is:** A self-supervised learning framework proposed by Yann LeCun where two inputs are encoded into a shared embedding space, and a predictor module learns to predict one embedding from the other. Unlike generative models, JEPA never reconstructs pixels — it operates entirely in latent space.

**Why it matters here:** LeWM is a JEPA: the encoder maps observations to latent embeddings, and the predictor forecasts next-step embeddings given actions. The entire contribution is about making JEPA training stable without heuristic tricks.

**Learn more:** [LeCun, "A Path Towards Autonomous Machine Intelligence" (2022)](https://openreview.net/pdf?id=BZ5a1r-kVsf)

## Representation Collapse

**What it is:** A failure mode in self-supervised learning where the encoder maps all inputs to the same (or nearly the same) representation, trivially satisfying any prediction objective. Also called "mode collapse" or "dimensional collapse" when only some embedding dimensions are used.

**Why it matters here:** Collapse is the central problem LeWM solves. Prior JEPAs use stop-gradients, EMA target encoders, or multi-term losses to avoid it. LeWM uses SIGReg instead, which provides a principled guarantee against collapse by enforcing distributional diversity.

**Learn more:** [Jing et al., "Understanding Dimensional Collapse in Contrastive Self-Supervised Learning" (ICLR 2022)](https://arxiv.org/abs/2110.09348)

## SIGReg (Sketched-Isotropic-Gaussian Regularizer)

**What it is:** A regularization technique that encourages high-dimensional embeddings to follow an isotropic Gaussian distribution. It works by projecting embeddings onto random 1D directions and applying the Epps–Pulley normality test to each projection. By the Cramér–Wold theorem, matching all 1D marginals implies matching the full joint distribution.

**Why it matters here:** SIGReg is one of the two loss terms in LeWM and is solely responsible for preventing collapse. Its key properties: scalable to high dimensions, only one effective hyperparameter (λ), and provable anti-collapse guarantees.

**Learn more:** [Balestriero & LeCun, "SIGReg" (2024)](https://arxiv.org/abs/2406.03229)

## Cramér–Wold Theorem

**What it is:** A classical result in probability theory stating that a multivariate distribution is uniquely determined by all of its one-dimensional projections (marginals). In other words, if two distributions agree on every 1D projection, they are the same distribution.

**Why it matters here:** Justifies SIGReg's approach — by enforcing Gaussianity along many random 1D projections, SIGReg effectively enforces that the full joint embedding distribution is Gaussian, without needing to test Gaussianity directly in high dimensions.

**Learn more:** [Wikipedia: Cramér–Wold theorem](https://en.wikipedia.org/wiki/Cram%C3%A9r%E2%80%93Wold_theorem)

## Model Predictive Control (MPC)

**What it is:** A control strategy where, at each time step, the agent optimizes an action sequence over a planning horizon using a predictive model, executes only the first few actions, then replans from the new state. This receding-horizon approach compensates for model inaccuracies.

**Why it matters here:** LeWM uses MPC at inference time — the learned world model rolls out candidate action sequences in latent space, and the Cross-Entropy Method (CEM) optimizes toward a goal embedding. The compact latent space makes this 48x faster than foundation-model alternatives.

**Learn more:** [Wikipedia: Model Predictive Control](https://en.wikipedia.org/wiki/Model_predictive_control)

## Adaptive Layer Normalization (AdaLN)

**What it is:** A conditioning mechanism where the scale and shift parameters of layer normalization are generated by an external signal (here, the action vector) rather than being learned as fixed parameters. This allows the network's behavior to be modulated by the conditioning input at each layer.

**Why it matters here:** The predictor uses AdaLN to incorporate action information at every transformer layer, initialized to zero so that action conditioning is introduced gradually during training.

**Learn more:** [Perez et al., "FiLM: Visual Reasoning with a General Conditioning Layer" (AAAI 2018)](https://arxiv.org/abs/1709.07871)
